Investigation: The Diagonal

When you draw a diagonal line from top-left to bottom-right, what can you say about the products on the line?

💡 Square Numbers! The diagonal passes through: 1, 4, 9, 16, 25, 36, 49, 64, 81, 100.
These are the square numbers! Each one is a number multiplied by itself (e.g. 3×3=9).
Also notice that the grid is symmetrical about the diagonal — 3×7 = 7×3. This is the commutative property!
3
Division Facts from the Square

Pick any product in the grid. Read across its row and up its column to find the two factors. Then write division facts!

Example: Product 24 → row 3, column 8 → so: 24 ÷ 3 = 8 and 24 ÷ 8 = 3

Why is multiplication commutative but division is not?

Write your explanation below. Try to use examples like 4×8 = 8×4 and 32÷8 ≠ 8÷32.

Hint: Multiplication is commutative because the order of factors does not change the product: 4×8 = 8×4 = 32. You can think of the array being rotated.

Division is NOT commutative because the order matters: 32÷8 = 4, but 8÷32 = 0.25 — completely different! The ÷ symbol means "share into groups," so the number being shared must come first.
📝 Summary of Key Ideas
✅ A multiplication square shows all products from 1×1 to 10×10.
✅ The diagonal contains square numbers: 1, 4, 9, 16 … 100.
✅ The square is symmetrical — mirror facts about the diagonal.
✅ Each product gives 4 facts (a fact family): 2 multiplication + 2 division.
✅ Multiplication is commutative; division is not.
